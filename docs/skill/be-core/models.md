# SKILL: MODEL (MONGOOSE)

## MỤC TIÊU

Chuẩn hóa cách tạo model mongoose.

## PHẠM VI

- schema
- index cơ bản

Không bao gồm:
- business logic

## QUY TẮC

model phải:

- định nghĩa field rõ ràng
- dùng type đúng
- dùng enum nếu cần
- dùng ref nếu có relation

model không được:

- chứa logic xử lý phức tạp
- chứa API logic

## KIỂM TRA

- field hợp lý
- naming đúng
- không dư field
