# Movie Booking App

Repo gom cac phan chinh:

- `mobile-app`: app nguoi dung React Native/Expo cho dang nhap, xem phim, dat ve, chon ghe va thanh toan.
- `admin-web`: web quan tri rieng. Hien tai moi la placeholder ban dau, chua dung UI library ben thu ba.
- `backend`: API dung chung cho mobile app, admin web va thanh toan.
- `fake-payment-gateway`: web thanh toan gia lap dung cho luong checkout.
- `docs`: tai lieu va ghi chu ky thuat.

## Chay app nguoi dung

```bash
cd mobile-app
npm install
npx expo start
```
